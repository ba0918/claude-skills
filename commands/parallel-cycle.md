指示を分解 → 並行 cycle 実行 → 自動マージのオーケストレーションを実行する。

## 使い方

- 自然言語で複合的な指示を渡す → 自動分解 → 並行実行
- 既存の plan ファイルパスを複数渡す → 分解スキップで並行実行

## 手順

1. スキル `claude-skills:parallel-cycle` を起動する
2. ユーザーの引数 `$ARGUMENTS` をそのまま渡す
3. 引数が空の場合は、何を実装したいかユーザーに質問する
